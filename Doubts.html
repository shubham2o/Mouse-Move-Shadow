HTML contenteditable Attribute

• The contenteditable attribute specifies whether the content of an element is editable or not.

• NOTE : When the contenteditable attribute is not set on element, the element will inherit it from its parent.

• The contenteditable attribute is a Global Attribute and can be used on any HTML element.

• EXAMPLE || An editable paragraph :

This is a paragraph. It is editable. Try to change this text.
